C-Data-Structures-Source-Code
A collection of fundamental Data Structures and classic algorithms implemented in the C programming language. Every file contains a complete, interactive program (with a menu-driven main function) ready to be compiled and run.

1. Stacks
Stack_using_array.c: Standard Stack implementation using a fixed-size array.

Stack_using_Linked_List.c: Standard Stack implementation using a Linked List.

Two_stack_in_one_array.c: An optimized method to run two separate stacks in a single array.

2. Queues
Queue_using_array.c: Linear Queue implementation using an array.

Queue_using_linked_list.c: Linear Queue implementation using a Linked List.

Circular_Queue_Using_Array.c: Circular Queue using an array, efficiently addressing space management.

Circular_Queue_Using_Linked_List.c: Circular Queue using a Linked List.

3. Linked Lists
Linked_List.c: Full implementation of a Singly Linked List (Create, CRUD, Search, Sort, Reverse).

Circular_Linked_List.c: Singly Linked List with circular connections.

Doubly_Linked_List.c: Doubly Linked List (forward and reverse traversal).

Doubly_Circular_Linked_List.c: Doubly Linked List with circular connections.

4. Trees & Algorithms
Binary_Search_tree.c: Implements BST creation, searching, and all three main traversals (Inorder, Preorder, Postorder).

infix_to_postfix.c: Algorithm to convert an Infix expression to a Postfix expression using a stack.
